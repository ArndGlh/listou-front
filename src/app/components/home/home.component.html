Listou Application web

Listou est une application web permettant aux utilisateurs de créer des évènements, y ajouter des participants et ensuite de gérer la liste d'achat à prévoir pour l'évènement. Chacun peut signaler aux autres ce qu'il compte acheter et dans quelles proportions, le prix que cela a coûté, etc. N'hésitez plus, inscrivez-vous sur Niceru pour en profiter !

Manga Update Application web

Manga Update est une application web permettant aux utilisateurs de créer une liste de mangas suivis, ajouter des urls de lecture en ligne, mettre à jour l'avancement du manga et d'être prévenu lorsque un manga sort un scan sur l'url renseigné. N'hésitez plus, inscrivez-vous sur Niceru pour en profiter ! (In progress)

AkwaKonJou Application web

AkwaKonJou est une application web permettant aux utilisateurs de créer une session de jeu et d'y ajouter d'autres utilisateurs. Une fois que tout les utilisateurs connectés ont renseigné leur(s) bibliothèque(s) de jeux (Steam, Epic, ...) AkwaKonJou affiche la liste des jeux en commun avec tout les participants, ou à défaut les jeux ayant le plus de joueurs. On peut alors les trier selon des filtres de genre, mode de jeu etc. (In progress)
